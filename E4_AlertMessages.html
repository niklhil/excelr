Alerts

Alerts are created with the .alert class, followed by a contextual color classes:

Success! This alert box indicates a successful action.
Info! This alert box indicates a neutral informative change.
Warning! This alert box indicates a warning.
Danger! This alert box could indicate a dangerous action.
Primary! Indicates an important action.
Secondary! Indicates a slightly less important action.
Dark! Dark grey alert.
Light! Light grey alert.
